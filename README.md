# 🇳🇵 Nepali Utility Suite

A multi-tool web application designed for Nepali students and users. This app provides a suite of high-precision tools in a single, modern, and responsive interface.

## ✨ Features

- **📅 Nepali Age Calculator**: Accurate conversion from Bikram Sambat (BS) to Gregorian (AD) and precise age calculation in years, months, and days.
- **🎓 +2 GPA Calculator**: Specifically designed for the NEB (National Examination Board) system, allowing users to calculate their GPA based on subject marks.
- **⏰ Digital Clock**: A high-precision, real-time digital clock with a clean, modern aesthetic.

## 🛠️ Tech Stack

- **Frontend**: HTML5, Tailwind CSS (Modern UI), JavaScript (ES6+).
- **Architecture**: Modular Client-Side Architecture.
- **Hosting**: Google Firebase Hosting.

## 🚀 Deployment

This project is automatically deployed to Firebase. To deploy updates:

```bash
firebase deploy
```

## 📁 Project Structure

```text
nepali-age-calculator/
├── public/
│   ├── index.html   # Main UI and structure
│   ├── style.css    # Custom theme and animations
│   └── script.js    # Modular tool logic
├── firebase.json    # Firebase hosting configuration
└── .firebaserc      # Firebase project alias
```

## 🛡️ Privacy & Security

This is a client-side application. All calculations are performed locally in the user's browser. No user data is sent to any server, ensuring 100% privacy.
